Basic linux commands:
mkdir folder_name    //to create a directory
cd folder_name	   //change directory
touch file_name	   //to create an empty file
ls			   //display all files
ls -a			   //list all files
vim file_name	   //to edit a file
i			   //to insert content
:wq			   //to save and quit
code .		   //to open files in vs code

Git Configuration
git config --global user.name "user_name"
git config --global user.email "your_email@example.com"
git config --global core.editor "code --wait"

Basic Git Commands:
git init		//to initialize a git repository
git status		//to display tracked, untracked files and changes. 
git add file_name	//adds file to the staging area
git add .		//adds all files
git commit -m "Commit message."  //to record the changes in the repository
git log 		//displays a history of everything that happens to a repository
git restore --staged file_name	//Removing changes from stage
git reset commit_id	//removes a commit from project history
rm -rf file_name		//removes a file
git stash			//store uncommited changes for later use
git stash clear		//clears stash
git stash pop		// re-apply the previous commits


Hosting a new project
git remote add origin url	//Connecting remote repository to local repository
git push origin master		//Pushing local changes to remote repository


Branches
git branch name			//create a new branch
git checkout branch_name	//switch to the branch
git merge branch_name		//merging branch to main
git push origin master		//pushing new changes to master branch 


Working with an existing project
git clone url			//cloning the forked project to local
git remote add upstream url	//upstream is from where you clone the repository
